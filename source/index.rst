IHChecklist
===========

Bem-vindo(a) a documentação do IHChecklist. Este projeto tem como objetivo
a criação um checklist prático de acessibilidade para sites, aplicativos e
plataformas digitais. Este é um projeto criado a partir do curso de
**Interação Humano-Computador**, ministrado pela nossa ilustríssima
professora `Rejane Maria de Costa Figueiredo
<https://www.pesquisar.unb.br/professor/rejane-maria-da-costa-figueiredo>`_.
